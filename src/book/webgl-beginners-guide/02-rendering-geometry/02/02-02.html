2.2. Overview Of WebGL's Rendering Pipeline

VBOs 中包含几何模型的数据, 包括: vertex coordinates, vertex normals, colors, texture coordinates

vertex shader 用来处理 VBOs 中的每一个数据, 并将这些数据表示为 vertex shader 上的一个 attribute, 每一个 attribute 都指向对应的 VBO

fragment shader 用来根据 vertex shader 中的数据, 计算每个像素的颜色

varyings 用来从 vertex shader 向 fragment shader 传递数据
